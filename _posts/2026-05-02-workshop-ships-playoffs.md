The Workshop announces the public release of **Playoffs**, a one-screen terminal dashboard for the NHL and NBA postseason. The product is implemented as a single Python file using only the standard library and is designed to fit within an 80-column terminal.

<!--more-->

Playoffs satisfies a longstanding operational requirement that the Founder be able to check sports results without opening a dedicated sports application, navigating an algorithmic feed, or being shown video preroll. The dashboard surfaces recent games, scheduled games, series standings, and active rounds. It does not surface advertising, video previews, opinion writing, or content the Founder did not request.

```
PLAYOFFS  Fri May 1 2026   updated 10:52 PM PDT

RECENT
  3h ago     NHL  R1 G6   VGK 5 @ UTA 1       VGK advances 4-2
  4h ago     NBA  R1 G6   LAL 98 @ HOU 78     LAL advances 4-2
  ...
```

Playoffs ships via GitHub at <code>github.com/maxspevack/playoffs</code>. The Workshop's domain holdings remain at zero (0); the subsidiary continues to operate without a public-facing web property.

The Workshop now operates two products in regular Founder use, satisfying the subsidiary's stated portfolio target.

> *Pursuant to the Workshop's standing operating principle: if it's longer than one file, it's a different subsidiary's problem. Playoffs fits in one file.*
